## 🚀 시작하기

1. 저장소 클론
```bash
git clone [repository-url]
cd algorithm
```

2. pre-commit 설치
```bash
pip install pre-commit
pre-commit install
```

3. 예제 실행
```bash
python Ch1/why_dsa_matters.py
```
